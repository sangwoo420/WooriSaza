## 프로젝트 소개

* 프로젝트명: 우리사자
* 서비스 특징 및 주요기능
  - 비슷한 지역에서 공동 구매를 통해 내가 필요한만큼, 더 싸게 구매하기!
  - 돈 관련하여 우리 사이트가 중간에서 물건을 수령하면 돈이 전달되도록 하여 안전성 확보하기!
  - 각 사용자마다 신뢰 등급, 댓글보기, 후기를 통해 사용자의 신뢰성 확인하기!
  - 서드 파티를 통한 계정 관리
  - 공동 구매를 희망하는 사람들끼리 파티를 맺을 수 있도록 플랫폼 제공
* 주요 기술
  - Vue
  - Spring boot
  - JPA
  - Rest API
